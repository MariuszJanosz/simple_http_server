## Simple http server
Simple http server written in C.

build
```
./build.sh [Debug | Release]
```

run
```
./run.sh
```
or
```
./build/simple_http_server <path_to_www_root> <ip> <port>
```

It requires .config file in the root directory of a website.
You would find information on syntax of .config in doc/syntax\_of.config.
